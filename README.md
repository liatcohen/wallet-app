# wallet-app

## Running the app

```
git clone https://github.com/liatcohen/wallet-app.git
cd wallet-app
```

To run the server:
```
cd server
npm install && npm run start
```


To run the client:
```
cd client
npm install && npm run start
```

Open http://localhost:3001 to view it in the browser.


## Running the tests
```
cd server && npm test
```
